# Blog App

A modern **blogging web application** built using **Next.js** and **MongoDB**. This project allows users to create, read, update, and delete blog posts with a responsive and user-friendly interface.

## 🚀 Features

* User authentication (optional, if implemented)
* Create, read, update, and delete blog posts (CRUD)
* Dynamic routing for blog posts
* Responsive design for all screen sizes
* Component-based architecture using Next.js
* Data storage and retrieval using MongoDB

## 🛠️ Technologies Used

* Next.js
* React.js
* MongoDB
* JavaScript
* HTML5
* CSS3
